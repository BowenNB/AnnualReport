## 使用

需要安装 node.js 版本 20

### 克隆代码

```bash
git clone https://github.com/LC044/AnnualReport

cd AnnualReport

cd single
```

### 安装依赖

```bash
npm install
```

### 运行

```bash
npm run dev
```

### 打包

```bash
npm run build
```
